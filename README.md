# Liar game
Web based liar game.

## Projects
```
├── liar-game-app           # Backend
├── liar-game-frontend      # Frontend
└── script                  # Deploy shell script
```

## Run server
```
cd liar-game-app
./gradlew clean bootJar
```

## Run front
```
npm install
npm start
```

## Environment
- Kotlin 1.5 (with Java 11)
- Gradle 7.2
- Spring Boot 2.6
- React 17
